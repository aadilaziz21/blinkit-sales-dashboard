# Blinkit Sales Dashboard (2023–2024)

##  Project Overview
This project is an interactive Microsoft Excel dashboard developed to analyze 12 months of Blinkit sales data. The dashboard provides business insights using Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting.

---

##  Objectives
- Analyze overall sales performance
- Track order status and delivery performance
- Understand customer purchasing behavior
- Monitor payment methods
- Build an interactive dashboard for business reporting

---

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

##  Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Delivery Performance
- Customer Spend Analysis
- Payment Method Distribution

---

##  Dashboard Features
- Interactive slicers
- Dynamic charts
- KPI cards
- Drill-down analysis
- Business insights
